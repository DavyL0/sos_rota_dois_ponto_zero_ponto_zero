# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-06-13

### Added
- CRUD completo de ocorrências com criar, editar e deletar
- Funcionalidade de cancelamento de ocorrências com justificativa
- Despacho de ambulâncias para atendimento de ocorrências
- Dashboard com visualização de dados em tempo real
- Tela de relatórios com análises e estatísticas

### Changed
- Melhorias na estrutura de componentes da interface
- Otimizações de performance na renderização de dados

## [1.1.0] - 2026-06-12

### Added
- Integração com PrimeNG para componentes UI avançados
- Suporte para arquitetura de projeto melhorada com estrutura de diretórios otimizada
- Setup inicial do projeto com estrutura base para frontend e backend
- Configuração de CI/CD com workflows do GitHub Actions para builds automatizados (frontend e backend)
- Arquivo .gitignore configurado para projetos Angular e Java

### Changed
- Reorganização dos diretórios do projeto para melhor estrutura
- Atualização de dependências com PrimeNG

### Fixed
- Melhorias na configuração inicial do projeto

## [1.0.0] - 2026-06-01

### Added
- Commit inicial do projeto
- Configuração base do repositório
- Setup inicial com arquivos de projeto
